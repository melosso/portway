## Development

You'll need to install **.NET 7** for `dotnet-tools` and **.NET 10 SDKs** for `Portway`.

```powershell
winget install --id Microsoft.DotNet.SDK.7 -e --silent --accept-package-agreements --accept-source-agreements
winget install --id Microsoft.DotNet.SDK.10 -e --silent --accept-package-agreements --accept-source-agreements
```
